# Extend with Powerful Tools (MCP)

The Model Context Protocol (MCP) lets ShunCode connect to external tools, databases, and APIs.

## What you can do

- **Query databases** directly from chat
- **Call APIs** and integrate external services
- **Discover tools** in the MCP Marketplace
- **Build custom servers** for your workflow

## Getting started with MCP

1. Open Settings → ShunCode → MCP Servers
2. Add a server configuration
3. ShunCode automatically discovers available tools

## Examples

- Connect to PostgreSQL for database queries
- Use GitHub MCP for repository management
- Add Figma MCP for design-to-code workflows
