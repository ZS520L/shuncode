# Start with a Goal, Not Just a Prompt

ShunCode works best when you describe **what you want to achieve**, not just what to type.

## How it works

1. Tell ShunCode your goal in natural language
2. It creates a plan and asks clarifying questions
3. Then it writes the code — like a true pair programmer

## Tips

- Be specific about the outcome you want
- Mention relevant files or technologies
- Let ShunCode ask follow-up questions

> "Build a REST API with authentication using Express and JWT"  
> is better than  
> "Write some Express code"
