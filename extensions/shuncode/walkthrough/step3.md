# Always Use the Best AI Models

ShunCode connects you to state-of-the-art AI models from leading providers.

## Supported providers

- **Anthropic** — Claude 4, Claude 3.5 Sonnet
- **Google** — Gemini 2.5 Pro, Gemini 2.5 Flash
- **OpenAI** — GPT-5, o3, o4-mini
- **And more** — DeepSeek, Groq, local models via Ollama

## Configuration

1. Open Settings → ShunCode → API Configuration
2. Add your API key for your preferred provider
3. Select the model that fits your needs

## Multi-provider support

Switch between providers instantly or configure multiple endpoints for different tasks.
